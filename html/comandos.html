Principais Tags HTML

<h1> até <h6> - Títulos

As tags <h1> a <h6> são usadas para definir títulos, do maior (<h1>) ao menor (<h6>).

<p> - Parágrafo

A tag <p> é usada para definir parágrafos de texto.

<a> - Links

A tag <a> cria links para outras páginas ou arquivos.

<ul>, <ol> e <li> - Listas

As tags <ul> (lista não ordenada), <ol> (lista ordenada) e <li> (itens da lista) são usadas para criar listas.

Exemplo de lista não ordenada:

  • Item 1
  • Item 2

Exemplo de lista ordenada:

  1. Passo 1
  2. Passo 2

<table> - Tabelas

A tag <table> cria tabelas com linhas (<tr>) e colunas (<td>).

Nome Idade
Ana 25
Pedro 30

<div> e <span> - Elementos de Agrupamento

A tag <div> é um container genérico para agrupar elementos.

A tag <span> é usada para estilizar partes específicas de um texto.

Exemplo: Este texto está vermelho.

<form> - Formulários

A tag <form> é usada para criar formulários com campos de entrada (<input>).
